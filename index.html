it is a function. Callback function are use to execute another function. 
        In JavaScript, a callback is a function that is passed as an argument to another function and is
        executed after some operation has been completed. Callbacks are commonly used in asynchronous programming,
         where a function may not complete immediately and the callback function is called when the operation is finished.
